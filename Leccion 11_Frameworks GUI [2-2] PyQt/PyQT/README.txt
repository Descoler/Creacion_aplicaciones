1. Crear entorno virtual:
    virtualenv PyQt_env
2. Instalar librerias de PyQt:
    sudo apt-get install qttools5-dev-tools
    sudo apt-get install qttools5-dev
3. Ejecutar editor: vamos a la carpeta de instalación de la libreria 
    - dirección: (/usr/lib/x86_64-linux-gnu/qt5/bin/)
    - Ejecutar:  ./designer
4. Para trabajar con pyqt5: (en el entorno virtual creado)
    pip install pyqt5
5. Para trabajar con PyQt6.
    pip install PyQt6
